# vue_ts
  基于Vue-cli 3.0框架搭建的前端工程文件

  引用库：
    TypeScript
    Vue-Router
    Vuex
    Stylus
    axios

  封装项：
    封装axios类库
    封装rem方法
    封装Vuex方法
    封装process.env.NODE_ENV方法
    封装Stylus全局使用方法
    按照个人习惯区分 "页面文件夹" "公共组件文件夹" "私有文件夹"

  额外封装：
    引用reset.css文件

## 启动服务器，baseUrl为 localhost：8080
```
yarn serve
```

### 启动服务器，baseUrl为 www.qzroc.com:8090
```
yarn start
```

### 打包生成文件
```
yarn run build
```
